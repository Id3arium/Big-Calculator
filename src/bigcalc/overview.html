Big Calculator Project Overview

In most programming languages, integer values are of a fixed size, e.g., 32 bits in C++, 64 bits in Java. 32 bits will allow integers of approximately 10 decimal digits. For most purposes this is sufficient, but when it is not, infinite precision integer arithmetic can be implemented in software. This project implements an infinite precision arithmetic package for integers.

Running the program

The input to this program is read from standard input and the output is directed to standard output.

The program is invoked from the command-line as: 

     java cisc187.bigcalc.{@link cisc187.bigcalc.Calc} < input-file
Where:
cisc187.bigcalc.Calc
is the program name
input-file
is the name of a file containing arithmetic sequences.
In this case, the results would be printed out to the console / terminal.

Alternative invocations are: 

     java cisc187.bigcalc.Calc < input-file > output-file 
     cat input-file | java cisc187.bigcalc.Calc 
     cat input-file | java cisc187.bigcalc.Calc > output-file 
     echo "2 2 +" | java cisc187.bigcalc.Calc
